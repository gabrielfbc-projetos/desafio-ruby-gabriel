# Desafio Automação - Gabriel Cartelli

## Stack

Ruby ≥ 2.6

Cucumber ≥ 3

Selenium ≥ 3.142

Chrome e chromedriver na última versão estável

## Instalação e Execução
Acesse a raiz do projeto via terminal e execute os comandos abaixo.
1. Instalar dependências:

    bundle install

2. Executar o teste 

    cucumber
